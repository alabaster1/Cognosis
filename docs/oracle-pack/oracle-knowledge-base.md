# Oracle Knowledge Base (Seed)

Date: 2026-02-18  
Status: starter template

Purpose:
- Keep source-of-truth references for consciousness, neuroscience, philosophy of mind, and parapsychology.
- Prevent citation drift and fabricated authority patterns.

## Use Rules
1. Prefer primary sources (papers, reviews, official datasets).
2. Distinguish replicated findings from exploratory results.
3. Mark source quality: high, medium, low.
4. Record known limitations and replication status.
5. If no strong source exists, explicitly mark evidence as weak/absent.

## Source Buckets
1. Mainstream neuroscience and consciousness science
- Example topics: NCCs, global workspace, IIT critiques, predictive processing

2. Psychology and cognition methods
- Example topics: bias, expectancy effects, preregistration, p-hacking controls

3. Parapsychology and anomalous cognition
- Example topics: Ganzfeld, remote viewing, presentiment, meta-analyses

4. Philosophy of mind
- Example topics: hard problem, panpsychism, physicalism, neutral monism

5. Statistics and methodology
- Example topics: Bayes factors, effect sizes, publication bias, meta-research

## Maintenance SOP
1. Add new sources weekly.
2. Re-check high-impact claims monthly.
3. Archive superseded or retracted sources.
4. Track date-sensitive updates with explicit calendar dates.
